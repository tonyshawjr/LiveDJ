# Changelog

All notable changes to LiveDJ will be documented in this file.

## [1.0.0] - 2025-12-27

### Initial Release

- Plex and Spotify support
- AI-generated artist, album, and song notes
- Multiple AI providers: OpenAI, Google Gemini, Groq (free)
- DJ voice intros with text-to-speech
- TTS providers: OpenAI, Groq (free), Browser (free)
- Beautiful display with album art and rotating slides
- Admin dashboard for configuration
- Smart caching (notes only generated once per artist/album)
- Auto-cleanup of TTS cache (7 days)
